Module 07: The Security Log

Audit Log

The Audit Log is the company’s "Security Camera." It records every important action taken by every user in the app. This makes sure there’s total transparency and accountability for every change, from hiring new people to processing payroll.

02. Search & Filter System

The Search Bar

You can search for anything—a user's name, a specific department, or even a detail like "Payroll Processed." The table updates instantly as you type.

Action Dropdown

To narrow things down, you can filter by Action Type. For example, if you only want to see "Leave Requests," you can select that from the dropdown to hide everything else.

04. The Data Export Engine

Download Audit Log (CSV)

This is a powerful feature that lets you download the entire history as a CSV file (which works in Excel). Unlike other "Export" buttons in the app, this one is fully functional!

The Code Logic (Blob)

We generate a "Blob" of text in the browser, create a temporary link, and "click" it automatically to start the download.

07. How to use this page

Investigating an event? Here’s how to use the Audit Log effectively:

STEP 1: USE THE SEARCH

Type a name or action (like "Payroll") into the search bar. This filters the list to only show matching rows.

STEP 2: FILTER BY ACTION

Use the dropdown to see only specific types of events, like "Leave Approved" or "Employee Created."

STEP 3: DOWNLOAD CSV

Click the Export CSV button at the top right to download the logs for offline analysis or auditing.

Privacy Note

Safety first! Only **Admin** and **HR** roles can see the Audit Log. Even Managers are restricted from this view, as it contains sensitive data about role changes and payroll processing.
