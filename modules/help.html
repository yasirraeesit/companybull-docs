04. Engineering Blueprint

Behind the Scenes (Code)

How we built the self-service hub:

Dynamic Categories

We use new Set() to automatically build the category filter buttons based on the data in faqItems—no manual list needed!

Multi-Open Logic

The Accordion is set to type="multiple", so you can open as many questions as you want at the same time to compare answers.

Deep Matching

The search logic is smart. It doesn’t just check the **Questions**; it also scans through every **Answer** to find the keyword you typed.

05. How to use this page

Need an answer fast? Here’s the best way to get help:

STEP 1: TYPE A KEYWORD

Instead of reading every question, type one word (like "Tax" or "Holiday") into the search bar for the fastest results.

STEP 2: FILTER BY TOPIIC

Use the **Category Buttons**. If you have a question about benefits, click the "Benefits" pill to hide everything else.

STEP 3: COMPARE ANSWERS

Click as many questions as you like. They will stay open so you can read through multiple answers at once.
