04. Engineering Blueprint

Behind the Scenes (Code)

Here’s the developer’s "magic" for the celebrations:

Smart Countdown

We use a function called getDaysUntil that handles wrapping dates (like if a birthday is in January and it's currently December).

Year Math

We calculate tenure by subtracting the joinYear from the current year, providing an accurate "X-Year" badge.

Avatar System

If an employee hasn't uploaded a photo, the AvatarFallback component generates a badge with their first and last initials.

Auto-Display

The "Today's Celebrations" card uses conditional rendering (&&) to stay hidden unless someone’s "celebration day" matches today's date.

// Anniversary Year Calculation Logic
years: today.getFullYear() - c.joinYear + (isCelebratedThisYear ? 0 : -1)
05. How to use this page

Want to spread some joy? Here’s the best way to use the Celebrations hub:

STEP 1: CHECK THE SPOTLIGHT

Check the top section every morning. If you see a name in the spotlight, reach out to them and say Happy Birthday or Congrats!

STEP 2: PRE-PLAN PARTIES

Use the "Upcoming" lists to see if anyone has a milestone (like 5 or 10 years) coming up next week so you can order a cake.

STEP 3: VIEW THE OVERVIEW

Check the monthly grid at the bottom to see every celebration for the whole month in one single glance.
