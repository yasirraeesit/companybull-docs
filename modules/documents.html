Module 12: The Secure Vault

Document Manager

The Document Manager is the company’s "Digital Filing Cabinet." It’s a secure place where all important paperwork—like contracts, legal agreements, and medical certificates—is stored safely. It ensures that the right people have access to the right files at the right time.

03. Action Center

Simple Upload

Adding a new file is easy. Click the **Upload** button at the top to add a document to an employee’s profile. (Currently simulates an upload for demo purposes).

Fast Download

Need a copy? Every file in the list has a small **Download** icon on the right. One click saves the file directly to your computer.

Controlled Delete

Mistakes happen. HR and Admins can delete documents using the red trash icon. Regular employees **cannot** delete files once they are uploaded.

04. Engineering Blueprint

Behind the Scenes (Code)

How we keep your files safe and organized:

Privacy Logic

We use a filter to check hasAccess("Admin", "HR"). If true, the user sees all files. If false, they only see files where the employeeId matches their own.

Dynamic Stats

The scoreboard counts are calculated in real-time using .length and .filter() on the document array.

Search Engine

The search bar is "Smart"—it checks both the **Filename** and the **Employee Name** at the same time.

Toast Feedback

Every action (Download, Delete, Upload) triggers a "Toast" notification at the bottom of the screen to confirm it worked.

05. How to use this page

Keeping your records tidy? Here’s the best way to manage your documents:

STEP 1: USE THE FILTERS

Instead of scrolling through hundreds of files, use the **Type Dropdown** to see only "Contracts" or "Certificates" instantly.

STEP 2: SEARCH BY NAME

(HR only) Type an employee’s name into the search bar to see every document associated with that specific person.

STEP 3: DOWNLOAD & REVIEW

Always verify the "Upload Date" before downloading to make sure you are looking at the most recent version of a contract.
