04. Engineering Blueprint

Behind the Scenes (Code)

How we built the forecasting engine:

Growth Pct Math

The system calculates ((EndOfYear - Current) / Current) * 100 to find the expansion percentage for the year.

Chart Layering

We use two overlapping Area components to show "Actual" (Solid) and "Forecast" (Dashed) data on the same chart.

Reference Line

A simple ReferenceLine component marks today's date, helping users distinguish between history and future plans.

Bar Chart Legend

The "BarChart" uses a horizontal legend so you can easily compare current staff vs. planned hires for each team side-by-side.

05. How to use this page

Planning for the next year? Here’s the best way to leverage Forecasting:

STEP 1: CHECK GROWTH %

If the growth % is too high (e.g., above 50%), you should talk to HR to make sure they have enough recruiters to handle the load.

STEP 2: FIND CRITICAL GAPS

Sort the Department Detail table by **Priority**. Focus on the red "High Priority" roles first for the current quarter.

STEP 3: CONSULT THE TIMELINE

Check the "Timeline" column for each role. If a role is marked for "Q3 2026," you can wait a few months before starting the search.
